Указание к использованию:

1. запросы на поиск добавляются в файл search.sh
2. чтобы запустить в консоли необходимо в каталоге решения прописать ./index.sh
3. если не будет запускаться, возможно в файле index.sh нужно поменять python3 на python



Описание решения:
Обычный булев поиск с обратным индексом, сжатым с помощью кода Фибоначчи.

для удобства в коде разделены комментариями отдельные блоки
1. сначала происходит чтение файлов
2. потом происходит парсинг файлов и разделение на отдельные страницы
3. затем прописаны функции кодирования и декодирования
4. происходит построение обратного индекса
5. для каждого документа происходит парсинг, построение дерева
6. далее прописана функция поиска
7. дальше просто происходит чтение из файла и обработка запросов пользователя
